## 📝 وصف المشروع (Project Description)
هذا المشروع عبارة عن نظام ويب تم بناؤه باستخدام إطار العمل **Laravel** ، ويحتوي على الأقسام المطلوبة للواجب مثل (اسم المشروع ملاذ لتنظيم ادارة مخيمات الايواء في غزة ). تم تطويره بهدف تطبيق مهارات إدارة المشاريع والتعاون باستخدام Git & GitHub.

---

## 🛠️ التقنيات المستخدمة (Technologies Used) [cite: 44]
* **Framework:** Laravel (PHP) 
* **Frontend:** Blade Templating Engine, HTML5, CSS3 
* **Version Control:** Git & GitHub

## خطوات التشغيل المحلي (Installation)
1. قم بعمل استنساخ للمستودع: `git clone`
2. تثبيت الحزم: `composer install`
3. إنشاء ملف البيئة: `cp .env.example .env`
4. توليد مفتاح التشفير: `php artisan key:generate`
5. تشغيل السيرفر المحلي: `php artisan serve`
